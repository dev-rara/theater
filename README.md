# Theater
**문화 예술 공연의 정보를 확인하고, 경험을 나누어 보세요!** <br>
날씨가 서늘해지니 음악회나 공연이 생각이 나는 것에 영감을 얻어 프로젝트를 진행했습니다.
<br>

## 🛠️기술 스택
* HTML, CSS, JS
* Python
* MongoDB
* Flask
* JWT<br>

## 💡구현 기능
* 회원가입
* 로그인, 로그아웃
* 아이디 중복확인, 아이디/비밀번호 형식 확인
* openApi(문화공연 정보)를 이용한 공연 정보 가져오기
* 공연정보를 카테고리별로 나누어 보여주기
* 공연정보에 대한 리뷰 등록하기
* 마이페이지에서 유저가 등록한 리뷰를 모아 보여주기
